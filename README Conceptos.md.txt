Como subir archivos al repositorio remoto: 
1.- Agregar los archivos nuevos a la termina con git add -A (si son todos) o solo el nombre del archivo que deseas subir
2.- Corroborar que esten leidos por la terminal con git status
3.- decir para que son los archivos a subir con git commit -m "comentario"
4.- finalmente a traves de git push subir al repositorio.
Es importante agregar el correo o usuario usado en GitHub para que este sea reconocido y subido correctamente.
